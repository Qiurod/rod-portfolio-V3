Selected Case Studies

產品開發主案例

這裡不只展示成品，而是把「為什麼設計、怎麼交付工廠、如何落地量產」攤開來看。作品集主軸從單張圖面升級為完整產品開發能力。

Case 01 / Gloves

Airflux M2
Concept to Production

以炎熱通勤情境為起點，將輕量網布、軟式拳眼護具與高辨識配色整合成夏季日常手套。此案完整呈現從概念稿、配色開發到量產實品的閉環。

  • Role產品概念、三視圖、CMF 配色、量產視覺對位
  • Decision用高透氣網布降低夏季悶熱，同時保留拳眼保護與騎士視覺辨識。
  • Result黑／紅／白三色上市，成為 SS25 手套系列銷售冠軍之一。

VALUE: 將通勤手套從「配件」提升為可被辨識、可量產、可銷售的系列單品。

Case 02 / Footwear

Gale C1 / Stratus C2
Urban Riding Shoes

鞋類系列的核心不是「做一雙像車靴的鞋」，而是把防水、護踝、換檔耐磨與日常街鞋輪廓壓在同一個產品語言裡，降低騎士從騎乘到生活場景的切換成本。

  • Role鞋型部件定義、CAD 圖面、技術模組轉譯、型錄視覺整合
  • Decision以 BOUSUI、atop、Ortholite 等模組建立可被消費者理解的技術敘事。
  • ResultGale C1 與 Stratus C2 形成透氣／防水雙路線，支撐 SS25 鞋類產品線。

VALUE: 從單品設計推進到產品線架構，讓鞋款能用同一套品牌語言被銷售。

Case 03 / Apparel

VELORA Riding Jacket
Spec Sheet to OEM Sample

機能服飾的關鍵在於圖面不能只好看，還要能被工廠讀懂。VELORA 以工程圖、材料製單、工法標註與實穿樣品串成完整開發鏈，讓版型、布料、五金與縫製細節可追蹤。

  • Role外套 CAD、材料規格製單、參考樣品標註、打樣溝通
  • Decision半網布結構對應夏季通風，反光印刷與可拆頭套補足騎乘安全與日常彈性。
  • Result從 CAD、Spec Sheet 到實穿樣品完成一致落地，降低工廠開版誤差。

VALUE: 不停在提案圖，而是把設計變成可被打樣、驗證、量產修正的開發文件。

SEC. 08

About

關於我

我是一名騎士裝備與機能產品設計師，現任 KOGENKI 品牌產品開發與創意主導。我的工作不是只產出好看的提案圖，而是把產品從市場分析、概念設計、CAD / Spec Sheet、樣品製單、OEM 溝通、打樣修正到量產驗貨完整推進。

我相信騎士裝備的好設計是「防護不能妥協，日常也要穿得住」。Protective gear 不該只停在賽道語言，它可以融入通勤、街頭穿搭與長途旅行，讓騎士不需要在安全與風格之間二選一。

我的優勢在於能同時理解產品結構、材料工法與品牌視覺：熟悉皮革、Clarino、防水織物、TPR / 碳纖護具等材料應用，能以 Illustrator / Photoshop 建立工廠可讀的圖面，也能延伸成型錄、包裝與電商主視覺，縮短從圖面到市場的距離。
